# imgGo

轻量级PHP图片处理类

* 支持生成缩略图
* 支持写入文件系统和返回原始数据流两种方式
* 支持缩略图格式转换和后缀自定义
* 完善的错误信息提示
* 支持生成随机文件名
* 支持返回Base64编码格式数据


## 更新记录
